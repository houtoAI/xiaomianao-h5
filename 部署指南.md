# 小棉袄项目部署指南

## 部署环境信息
- 域名：xiaomianao.love
- 需求：HTTPS（语音功能必须）
- 每日对话限制：20次/用户

---

## 一、阿里云服务器基础配置

### 1. 登录阿里云控制台
1. 打开浏览器访问：https://www.aliyun.com
2. 登录你的阿里云账号
3. 进入"云服务器ECS"控制台

### 2. 获取服务器信息
1. 找到你的ECS实例
2. 记录以下信息：
   - 公网IP地址
   - 操作系统版本（推荐Ubuntu 20.04或CentOS 7+）
   - SSH登录端口（默认22）

### 3. 安全组配置（重要）
1. 进入实例详情 → 安全组
2. 配置入方向规则，开放以下端口：
   - **80端口**（HTTP）
   - **443端口**（HTTPS）
   - **22端口**（SSH）
   - **3000端口**（Node.js服务，仅调试用）

## 六、配置Nginx和HTTPS

### 1. 安装Nginx
```bash
# Ubuntu
sudo apt install -y nginx

# CentOS
sudo yum install -y nginx
```

### 2. 申请SSL证书
阿里云提供免费SSL证书：

1. 进入阿里云控制台 → SSL证书（应用安全）
2. 点击"购买证书"
3. 选择"免费证书"（DV SSL）
4. 填写域名：xiaomianao.love
5. 验证域名所有权（添加DNS解析记录）
6. 等待证书签发（约5-10分钟）
7. 下载证书（选择Nginx格式）

### 3. 上传SSL证书
```bash
# 创建证书目录
sudo mkdir -p /etc/nginx/ssl

# 上传证书文件（在本地执行）
scp xiaomianao.love.pem root@服务器IP:/etc/nginx/ssl/
scp xiaomianao.love.key root@服务器IP:/etc/nginx/ssl/
```

### 4. 配置Nginx
创建配置文件：
```bash
sudo nano /etc/nginx/sites-available/xiaomianao
```

写入以下内容：
```nginx
# HTTP重定向到HTTPS
server {
    listen 80;
    server_name xiaomianao.love;
    return 301 https://$server_name$request_uri;
}

# HTTPS配置
server {
    listen 443 ssl;
    server_name xiaomianao.love;

    # SSL证书
    ssl_certificate /etc/nginx/ssl/xiaomianao.love.pem;
    ssl_certificate_key /etc/nginx/ssl/xiaomianao.love.key;

    # SSL优化
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;

    # 前端静态文件
    location / {
        root /root/xiaomianao-h5;
        index homepage.html;
        try_files $uri $uri/ /homepage.html;
    }

    # API代理
    location /v1/ {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }

    location /api/ {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
# Ubuntu
sudo ln -s /etc/nginx/sites-available/xiaomianao /etc/nginx/sites-enabled/
sudo nginx -t  # 测试配置
sudo systemctl restart nginx

# CentOS
sudo nginx -t
sudo systemctl restart nginx
```

## 八、域名解析配置

### 1. 配置DNS解析
阿里云控制台 → 域名 → 解析设置：

添加两条A记录：
- **记录类型**：A
- **主机记录**：@（主域名）
- **解析路线**：默认
- **记录值**：你的服务器公网IP

- **记录类型**：A
- **主机记录**：www
- **解析路线**：默认
- **记录值**：你的服务器公网IP

### 2. 等待解析生效
DNS解析通常5-10分钟生效，最多24小时

## 十、常见问题解决

### 问题1：语音功能无法使用
**原因**：浏览器要求HTTPS
**解决**：必须配置SSL证书，确保https://xiaomianao.love访问

### 问题2：麦克风权限被拒绝
**原因**：浏览器安全策略
**解决**：确保HTTPS，并在浏览器中允许麦克风权限

### 问题3：服务启动失败
```bash
# 查看错误日志
pm2 logs xiaomianao-api --lines 100

# 检查端口占用
sudo netstat -tulpn | grep :3000

# 手动启动测试
cd /root/gemma-api-server
node server.js
```

### 问题4：域名无法访问
```bash
# 检查DNS解析
nslookup xiaomianao.love

# 检查Nginx状态
sudo systemctl status nginx

# 检查防火墙
sudo ufw status  # Ubuntu
sudo firewall-cmd --list-all  # CentOS
```
